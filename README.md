# Retail Inventory Analytics System

## Project Overview
A Retail Inventory Analytics System developed using Spring Boot, Java, MySQL, HTML, CSS, and JavaScript. The application helps manage retail inventory by adding, viewing, and tracking products.

## Technologies Used
- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript
- Maven

## Features
- Add Products
- View Product List
- Inventory Management
- Responsive User Interface
- CRUD Operations

## How to Run
1. Clone the repository.
2. Configure MySQL in `application.properties`.
3. Run:
4. mvn spring-boot:run
5. Open:
6. http://localhost:8080/products
